# Mooseville AI Generator Prompt Export

Purpose: Copy-paste prompt blocks for AI 3D generation, aligned to Mooseville style and production needs.

## Global Prompt Prefix

Use this prefix at the start of every prompt:

Stylized rustic-fantasy moose town asset, handcrafted timber and stone, antler-safe architecture, hoof-friendly proportions, carved moose motifs, readable third-person silhouette, game-ready topology, modular parts, clean UVs, PBR textures.

## Global Negative Prompt

Use this as shared negative guidance:

No modern city props, no sci-fi panels, no human logos, no realistic humans, no tiny doors, no sharp hazardous corners, no photoreal mud noise overload, no broken topology, no non-manifold meshes, no floating disconnected geometry.

## Recommended Output Settings

- Format: FBX and GLB
- Scale: meters, character height reference 2.1m Moosian
- Pivot: centered at ground contact
- LODs: LOD0 hero, LOD1 gameplay, LOD2 background
- Texture targets: 2K hero, 1K common
